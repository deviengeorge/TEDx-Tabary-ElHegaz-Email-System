## Run Locally

Clone the project

```bash
  git clone https://github.com/deviengeorge/TEDx-Tabary-ElHegaz-Email-System
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start The System

```bash
    npm run start
```
